============================================
Crouching Alchemist Hidden Panda (CALCHIPAN)
============================================

What is it?
===========

A `SQLAlchemy <http://www.sqlalchemy.org/>`_ dialect which will
consider a series of `Pandas <pandas.pydata.org/>`_ DataFrame objects
as relational database tables.
